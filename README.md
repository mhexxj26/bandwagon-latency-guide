## 搬瓦工 200 延迟是什么概念？

**延迟（Ping 值）**：你的电脑发一个数据包到服务器，再收到回应，这个来回的时间，单位是毫秒（ms）。延迟越低，操作越跟手，尤其是 SSH 登录、实时通信类的场景感知最明显。

200ms 在国内访问美国服务器这个语境里，算一个分界线：低于这个数字，体感基本流畅；高于这个数字，SSH 打字会有轻微的卡顿感，视频流也更容易缓冲。所以搬瓦工 200 延迟放在什么机房什么线路上，结论完全不一样。
